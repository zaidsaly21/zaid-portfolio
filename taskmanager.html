Task Manager App

Full-stack task management application

React Node.js Express MongoDB Mongoose

A full-stack task management application built to practice end-to-end development, from database design to a working user interface. The frontend is built with React, organized into reusable components (TaskForm, TaskList, TaskItem), each with its own styling. The backend is a Node.js and Express REST API, with MongoDB and Mongoose handling data storage and schema validation.

What it does

  • Create, view, update, and delete tasks (full CRUD functionality)
  • Each task includes a title, description, category, completion status, and optional due date
  • Automatic timestamps for when tasks are created and updated

How it was built

  • React components for the form, task list, and individual task items, keeping the UI modular and easy to maintain
  • Express REST API with separate routes for GET, POST, PATCH, and DELETE operations
  • MongoDB schema design using Mongoose, with sensible defaults and built-in timestamps
  • API endpoints tested using Thunder Client before connecting them to the frontend
  • Environment variables used to securely store the database connection string

What I learned

  • How a React frontend communicates with a backend through REST API calls
  • Structuring an Express API with clean, organized routes and proper error handling
  • Designing a MongoDB schema and understanding how Mongoose models map to it
